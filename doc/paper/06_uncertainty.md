# 不確かさ（統計＋系統）

本論の各検証には、（A）一次データのばらつきに由来する **統計的不確かさ** と、（B）観測系・モデル化・前処理に由来する **系統不確かさ** が同時に含まれる。  
ここでは、各検証で「何が支配的か」と「どこまで主張してよいか」を表にまとめ、**この表の範囲外の主張はしない**（過剰な一般化を避ける）。

| テーマ | 統計（代表指標） | 主な系統要因（例） | 本論での主張範囲 |
|---|---|---|---|
| LLR（月レーザー測距） | inlier RMS（観測−モデル）：<br>SR=8.914 ns / SR+Tropo=4.494 ns / SR+Tropo+Tide=4.281 ns | 局座標/EOP、対流圏（マッピング関数）、潮汐/海洋荷重、反射器ラベル混入、装置・解析系の更新 | ns級での整合性と寄与分解（対流圏・潮汐）の「効き方」を示す。外れ値は統計から除外し、一次データ行へ追跡できる形で提示する。 |
| Cassini（太陽会合・ドップラー） | ±10日：RMSE=7.069e-11 / corr=0.96308 | TDFの量（観測−予測の疑似残差）の定義、参照Shapiroの足し戻し、プラズマ補正、欠測区間の扱い、時間原点の整合 | 一次データを用いた「形状とスケールの整合性チェック」。β拘束は強く主張せず、処理前提の明示を優先する。 |
| Viking（太陽会合・Shapiro遅延） | 文献代表値（ピーク）：概ね 200–250 μs | 生データ時系列ではなく文献代表値、幾何（暦）依存、電波観測系の詳細非同定 | 係数スケールとピーク位置の整合性チェック（一次データ時系列フィットは別課題）。 |
| Mercury（近日点移動） | 推定：42.992913 ″/世紀（代表値42.98 ″/世紀との差 0.012913 ″/世紀） | 他惑星摂動、太陽四重極、観測残差の定義、初期条件/積分誤差 | 「弱場での補正項のスケール」確認として提示し、精密暦の完全再現は主張しない。 |
| GPS（衛星時計） | 2025-10-01（31衛星）：中央値RMS<br>BRDC=0.3278 ns / P-model=0.4022 ns<br>周期成分：corr=0.999818 / RMSE=0.4776 ns | 軌道・時計推定、受信機側系統、電離層/対流圏、アンテナ位相中心、補正体系（dt_relの扱い） | 「周期成分の一致度」と「同一入力で再現可能な比較枠」を主張の中心に置く（単一指標で優劣を断定しない）。 |
| EHT（ブラックホール影） | κ=1仮定の整合性（zスコア）＋κ_fit | κ（リング/影）、放射モデル、散乱、画像再構成、Kerrスピン/傾斜角での影定義レンジ | 強い“証明”ではなく整合性チェック＋差分予測の入口（系統誤差が支配的であることを明示）。 |

参考（系統要因の典型例）：LLRはIERS/測地モデル・対流圏モデル、EHTはリングと影の対応（κ）とKerrスピン依存が支配的である。[IERS2010][Saastamoinen1972][Niell1996][EHTM87I2019][EHTSgrAI2022]
