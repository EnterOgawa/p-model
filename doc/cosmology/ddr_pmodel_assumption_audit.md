# DDR前提監査（Step 5.3.2）: FRW前提リストと P-model（Part I 2.5）の照合

目的：`doc/cosmology/ddr_frw_assumptions_ledger.md`（Step 5.3.1）の前提リストを、
プロジェクト内の既存定義（Part I 2.4–2.5）と照合し、
P-modelで **維持（keep）/破棄（break）/置換（replace）**の判定を固定する。

この監査は「DDRがどこで破れるか」を論点として閉じるための台帳であり、
距離指標の救済（再接続；α, s_L, s_R の最適化）そのものは Step 4.7 / Step 5.3.6 で扱う。

---

## P-model側で先に凍結されている定義（参照：Part I）

本プロジェクトでは、宇宙論（背景P）について少なくとも次を凍結済みとして扱う。

- 背景分解：`P(x,t)=P_bg(t) P_local(x)`
- 赤方偏移（周波数比）：`ν_obs/ν_em = P_obs/P_em` → `1+z = P_em/P_obs`
- 時間伸長（p_t）：条件A（自由伝播で dθ/dt 保存）を採用する限り、`Δt_obs=(1+z)Δt_em`（p_t=1）

この監査では、上の凍結定義を優先し、勝手に再定義しない。

## 監査（keep / break / replace）

以下、Step 5.3.1 の各前提に対する判定。

### 1) 観測量の定義（D_L, D_A, z）

- 判定：**keep（定義としては維持）**
- 理由：
  - `D_L` と `D_A` は「観測量の定義」であり、理論側の座標写像とは別物として固定できる。
- 注意（重要）：
  - 実務上は、SNe（標準光源）や BAO（標準定規）から `D_L`/`D_A` を推定する段階で
    多数の前提（標準化、校正、系統、共分散、fiducial cosmology）が入る。
  - よって「定義は keep」だが「推定手順は前提依存」であり、比較は Step 4.7/5.3.6 の枠で行う。

### 2) FRW metric（等方・一様、空間膨張 a(t)）

- 判定：**break**
- 理由：
  - P-model（背景P）は「空間が伸びる」仮定を必須にしない（赤方偏移は P_bg の時間変化で説明する）。
  - したがって FRW 固有の `D_A = D_M/(1+z)` を支える前提（空間膨張によるスケール因子）は採用しない。

### 3) 光の伝播が null geodesic（FRW-2）

- 判定：**replace（形式は維持するが内容は置換）**
- 理由：
  - P-modelでは光路を屈折率 `n(P)` とフェルマー原理で記述する（局所Pの勾配で曲がる）。
  - これは「光線近似（幾何光学）」という枠組み自体は keep だが、伝播則は FRW の最小形（null geodesic）とは異なる。
- 条件：
  - 媒質が損失なし（透過）で、位相追跡が成立するなら、距離の定義（dΩ↔dA）や表面輝度保存に相当する性質は残り得る。
  - 損失や強い散乱がある場合は、DDRは伝播側で破れ得る（不透明度 α の解釈へ接続）。

### 4) 赤方偏移が 1+z=a0/a_em（FRW-3）

- 判定：**replace**
- 理由：
  - P-modelでは `1+z=P_em/P_obs` を凍結している。
  - したがって「zをa(t)の比で与える」FRW固有の前提は採用しない。

### 5) 角径距離で D_A = D_M/(1+z)（FRW-4）

- 判定：**break**
- 理由：
  - この `1/(1+z)` は「空間膨張で、放射時刻の物理長さが縮む」ことに由来する。
  - P-model（空間膨張なし）では、同じ幾何（静的）を採る限り `D_A` にこの因子は入らない。
- 置換の形（例）：
  - 静的・平坦の最小：`D_A^(P) = D`（D は静的幾何の距離）
  - 距離 D(z) の関数形（例：指数P_bgで `D∝ln(1+z)`）は別途だが、DDR比そのものとは分離できる。

### 6) フラックスで 1/(1+z)^2 が入る（FRW-5）

- 判定：**keep（条件付き）**
- 理由：
  - エネルギー比 `E_obs/E_em = ν_obs/ν_em` と、到着率 `dt_obs/dt_em` が赤方偏移に結びつく、
    という構造自体は P-modelでも採用できる。
  - とくに、条件A（dθ/dt保存）+ 時計写像（dτ/dt=P0/P）により p_t=1 が固定されるなら、
    到着率は `1/(1+z)` となる。
- 条件：
  - 光子数保存（不透明度なし）を仮定する。
  - 周波数比とエネルギー比を同一視できる（散乱・分散等で破れない）。

## 結論（この監査の固定点）

P-model（空間膨張なし・背景Pで赤方偏移）で、かつ

- 光子数保存（opacityなし）
- エネルギー比 `E_obs/E_em = 1/(1+z)`
- time dilation `dt_obs/dt_em = (1+z)`（p_t=1）

を採用すると、FRWで `(1+z)^2` を作る要素のうち
`D_A = D_M/(1+z)` の `1/(1+z)` が消えるため、DDRは

- `D_L^(P) = (1+z) D_A^(P)`

へ **必然的にずれる**（詳細導出は Step 5.3.3–5.3.5）。

このズレを観測（ε0）と整合させるには、距離指標前提（標準光源/標準定規/不透明度など）側で
追加の自由度（α, s_L, s_R など）を明示して検証する必要がある（Step 4.7 / Step 5.3.6）。
