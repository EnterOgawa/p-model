# DDR前提台帳（Step 5.3.1）: FRWで D_L=(1+z)^2 D_A を得るまで

目的：標準宇宙論（FRW）で距離二重性（distance duality relation; DDR）

- `D_L = (1+z)^2 D_A`

が導かれるまでの各ステップを追い、**そこで使われる前提**を台帳化する。

このメモは **Phase 5 / Step 5.3.1** の成果物である（P-model側の維持/破棄は Step 5.3.2）。

---

## スコープ

- ここでは「導出の前提の棚卸し」だけを行い、宇宙論全体の救済（BAO/H0/距離指標の再接続）は扱わない。
- ただし「どの前提が破れるとDDRが破れ得るか（破れ方の分類）」は、前提の整理として短く含める。

---

## 観測量の定義（出発点）

DDRは「距離の定義」を含むため、まず定義を固定する。

- **赤方偏移**：`1+z ≡ ν_em/ν_obs`（周波数比）
- **光度距離**：`F_obs ≡ L_em / (4π D_L^2)`（bolometric flux と bolometric luminosity）
- **角径距離**：`D_A^2 ≡ dA_em / dΩ_obs`（放射側の固有面積要素と観測側の立体角）

前提（定義に含まれるもの）：

- L_em は「放射体の静止系（固有時）での光度」として定義する（観測側の前処理で割り戻した量ではない）。
- F_obs は「観測側の検出器面での受光エネルギー/単位観測時刻/単位面積」として定義する。
- dA_em, dΩ_obs の定義に「どの波長帯を同一視するか（K補正等）」や「レンズで複数像があるときの取り扱い」など、
  実務上の選別が入る（導出の数学以前に、観測処理が前提依存である）。

---

## FRW導出（典型ルート）と前提

ここでは「FRWの式を順に追う」最小ルートを採り、各行で使う前提を列挙する。

### FRW-1: 幾何（FRW metric を採用）

導出で使う典型形：

- `ds^2 = -c^2 dt^2 + a(t)^2 [ dχ^2 + S_k(χ)^2 dΩ^2 ]`

前提：

- （FRW仮定）宇宙の大域が「等方・一様」で、時刻関数 a(t) で空間スケールが決まる。
- （metric）時空が（少なくとも光の伝播に関して）計量で記述される（Riemann幾何；非計量・ねじれ等は採らない）。
- （comoving）観測者・放射体が宇宙流体に静止（peculiar velocity をここでは無視）している。

### FRW-2: 光の伝播（radial null geodesic）

- `ds^2=0`（光）から `dt/a(t)` と χ の関係を得る。

前提：

- （geometric optics）波としての光を「光線」として扱える（波長が曲率スケール等に比べ十分小さい）。
- （null propagation）光線は null geodesic に沿う（散乱・分散・媒質による非測地線は無い）。
- （single path）source→observer の対応が一意（強レンズで複数経路が支配的な場合は、後段の dΩ の定義が実務上あいまいになる）。

### FRW-3: 赤方偏移（1+z = a0/a_em）

- `1+z = a(t_0)/a(t_em)`

前提：

- （expansion redshift）波長伸長が a(t) の比で与えられる（宇宙膨張が主因）。
- （clock mapping）観測者の時刻は FRW の宇宙時刻 t に沿って比較できる（重力ポテンシャル差等は無視）。

### FRW-4: 角径距離（D_A）

FRWでは、放射時刻 t_em の「物理的な横サイズ」が `a(t_em) S_k(χ)` で与えられるため、

- `D_A = a(t_em) S_k(χ) = D_M/(1+z)`（D_M は transverse comoving distance）

前提：

- （physical length）固有長さは a(t) 倍でスケールする（空間が伸びる）。
- （small-angle）横サイズと角度の関係 `ℓ = D_A · δθ` を用いる（天体サイズが十分小さい等）。
- （no extra optics）観測系のPSF/散乱/屈折など、角度の定義を崩す系統は別扱いとする。

### FRW-5: フラックス（F_obs）

等方放射（bolometric）を仮定し、観測半径の球面の面積 `4π D_M^2` に広がるとして、

- 面積因子：`1/(4π D_M^2)`
- 光子エネルギー低下：`E_obs/E_em = 1/(1+z)`
- 到着率の低下（time dilation）：`(dt_obs/dt_em) = (1+z)`

より、

- `F_obs = L_em / [4π D_M^2 (1+z)^2]`

前提：

- （isotropic emission）放射が等方（少なくとも「等価的に」等方として L_em を定義できる）。
- （photon number conservation）光子数が保存（吸収・散乱・光子↔別粒子変換などが無視できる）。
- （energy redshift）各光子のエネルギーが `1/(1+z)` で減衰（周波数比と一致）。
- （time dilation）到着率が `1/(1+z)` で減衰（放射側と観測側の時間の対応が z と結び付く）。

### FRW-6: 光度距離（D_L）とDDR

定義 `F_obs ≡ L_em/(4π D_L^2)` により、

- `D_L = (1+z) D_M`

一方 `D_A = D_M/(1+z)` なので、

- `D_L = (1+z)^2 D_A`

前提：

- ここまでの前提の合成（特に FRW-4 の「空間スケール a(t)」と、FRW-5 の「(1+z)^2 の減衰」）。

---

## どの前提が破れるとDDRが破れ得るか（分類）

DDRが破れる典型の分類（「何が変わるとD_L/D_Aが変わるか」）：

- **A: 光子数が保存されない**（opacity, scattering, photon–axion conversion 等）
  - FRW-5 の photon number conservation が破れる → D_L 側が主にずれる。
- **B: 光が null geodesic をたどらない**（媒質・分散・屈折を含む伝播則の変更）
  - FRW-2 が破れる → dΩ と dA の関係や、エネルギー/到着率の関係が変わり得る。
- **C: 幾何がFRWではない（空間膨張がない等）**
  - FRW-1/FRW-4 が破れる → `D_A = D_M/(1+z)` の `1/(1+z)` が消える（典型）。
- **D: 観測処理（標準化/校正/選別）に z を埋め込む**
  - 定義（観測量の推定）自体が前提依存になり、理論比較が循環し得る（距離指標前提の監査が必要）。

---

## まとめ（台帳として固定できたこと）

- FRWで `D_L=(1+z)^2 D_A` が出る決定点は、
  - `D_A = D_M/(1+z)`（空間膨張に由来）と
  - `F_obs ∝ 1/(1+z)^2`（エネルギー×到着率の減衰）
  の **両方**を同時に採用している点にある。
- 後段（P-model側）の作業は、
  - どの前提を維持し、どれを破棄/置換するか
  を明示して、P-model固有のDDR関係を固定する（Step 5.3.2〜）。
