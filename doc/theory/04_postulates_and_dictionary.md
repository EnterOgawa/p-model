## 0. 統一コンセプト（このプロジェクトの合言葉）

- **質量近傍で P は増える**
- **波は“最短経路”を好む**（ただし最短＝幾何学的距離ではなく、波にとっての距離）
- **P 勾配へ滑り落ちる＝重力**
- **光も P に寄る＝屈折（曲がり）**

この4つを、以後すべての説明・式・検証で“同じ意味”で使う。

## 1. 辞書（記号と意味）

### 1.0 t と τ（時間の2種類）
- `t`：外部基準時（遠方基準系での時間、観測データの時刻の基準）
- `τ`：固有時（物体自身の時計が刻む時間）

このプロジェクトでは、時計の進み（固有時の増え方）を
```
dτ/dt = (P0/P) * (dτ/dt)_v
```
と分けて扱う：
- `P0/P`：重力（P）項（質量近傍で遅くなる）
- `(dτ/dt)_v`：速度項（SR相当。Pとは独立に書ける）

### 1.1 P（時間波密度）
- `P(x)`：空間点 `x` における **時間波の密度**（“刻みの細かさ”）。
- `P0`：基準値（遠方・無限遠・重力源から十分離れた領域の `P`）。
- 重要：本プロジェクトでは **質量近傍で `P>P0`** とする。

### 1.2 φ（ニュートン形ポテンシャルへの写像）
P-modelでは、ニュートン形（無限遠で0、質量近傍で負）へ合わせるために
```
φ ≡ -c^2 ln(P/P0)
```
を定義する。

この定義により：
- `P>P0`（質量近傍）なら `φ<0`
- 弱場で `P/P0 = exp(GM/(c^2 r))` を採ると `φ = -GM/r` になる

また、静止時計（v=0）の重力項は
```
(dτ/dt)_g = P0/P = exp(φ/c^2)  ≈ 1 + φ/c^2
```
となる（弱場 `|φ|/c^2 << 1`）。

### 1.3 重力（運動方程式）
重力は「P勾配へ滑り落ちる」として、ニュートン形に統一する：
```
a = -∇φ = c^2 ∇ln(P/P0)
```

### 1.4 光（屈折：高P側へ曲がる）
光（自由波）は、Pにより“実効屈折率”が決まると置く：
```
n(P) = (P/P0)^(2β)   （β>0）
c_eff(P) = c / n(P)
```
PPN mapping: (1+gamma) = 2*beta  (gamma = 2*beta - 1)
ここで `c` は `P=P0` のときの基準光速（遠方での光速）とする。

#### 1.4.1 係数2は「βが担う」（√ を φ に入れない）
- Shapiro遅延や光の偏向の標準形は PPN で `(1+γ)` が係数として現れ、GR では `γ=1` なので `(1+γ)=2` になる。
- P-model ではこの係数を **光伝播側の指数 `β`** で担保する：`(1+γ)=2β`。  
  したがって **GR整合は `β=1`**（Cassiniの拘束により `β≈1` に強く固定される）。
- 重要：この係数2を合わせるために `φ` の定義を `φ=-c^2 ln(sqrt(P/P0))` のように変えない。  
  `φ` はニュートン形（`φ=-c^2 ln(P/P0)`）に固定し、観測に対する微調整は `β`（光セクター）のみで行う。

### 1.5 「最短経路」の意味（最重要）
このプロジェクトでいう「波が最短経路を好む」は、次の意味に統一する：
- 幾何学的距離 `∫ ds` の最小ではない
- **光学経路長（または到達時間）**の停留（多くの場合は最小）として扱う

光なら、フェルマーの原理として
```
T = (1/c) ∫ n(P(x)) ds  を停留させる経路
```

補足（「経路長」と「波面密度」）：
- 一定周波数 `ω` の波では、位相の増分は `dφ_phase = k ds`（`k = n ω / c`）。
- したがって `∫ n ds` は「到達時間」だけでなく「位相の蓄積（波面密度）」とも直結している。
- この意味で、P-modelの「波が最短経路を好む」は **経路長（到達時間/位相）**の最適化であり、振幅（大きさ）そのものの最適化ではない。

その結果、`n`（したがって `P`）が空間的に変化すると光路は曲がり、
`β>0` なら **高P側（高n側）へ屈折する**。

### 1.6 δ（速度項の飽和：普遍定数）
本プロジェクトでは「波の特性として 100%（無限大/完全）はあり得ない」という考えから、
速度（SR相当）の効果は **光速極限で発散せず飽和**すると置く。

- `δ>0`：飽和を与える **普遍定数**（ただし **極小**）
- 速度項（検証で使う結論形）：
```
(dτ/dt)_v = sqrt( (1 - v^2/c^2 + δ) / (1 + δ) )
γ_δ(v) ≡ dt/dτ = sqrt( (1 + δ) / (1 - v^2/c^2 + δ) )
γ_max = lim_{v->c} γ_δ(v) = sqrt((1+δ)/δ) ≈ 1/sqrt(δ)
```

注意：
- 低速では `δ<<1` なら SR とほぼ同じ（差が出るのは `1 - v^2/c^2 ~ δ` の領域）
- 既存の高エネルギー観測で巨大な `γ` が得られているため、`δ` は非常に小さくなければならない  
  例：`γ_max >> 10^4` を要求するなら `δ << 10^-8`、`γ_max >> 10^11` なら `δ << 10^-22`
- 運用上の暫定採用値（余裕を見た安全側）：`δ = 1e-60`（`γ_max ~ 1e30`）  
  ※ 太陽系スケール（GPS/Cassini等）の検証では、`δ` の影響は実質的に現れない。

## 2. 公理（最小セット）

以後の構築は、まずこの公理セットで進める（必要が出たら改訂するが、改訂時はこのファイルを更新する）。

### Postulate A：物質＝波
- すべての物質は波（束縛波）として表現される。
- 光は自由波として表現される。

### Postulate B：時間波密度 P の存在
- 空間にはスカラー場 `P(x)` があり、局所の“時間の刻み”を規定する。

### Postulate C：質量は P を増やす
- 質量（より一般にはエネルギー密度）は周囲の `P` を増やす（`P>P0`）。

### Postulate D：重力＝P勾配への滑り落ち
- 物質（束縛波）は `P` の勾配により運動が偏り、巨視的には
  `a = c^2 ∇ln(P/P0)` として現れる。

### Postulate E：光の伝播＝Pにより屈折
- 光は `n(P)=(P/P0)^(2β)`（`β>0`）に従い、光学経路長の停留条件で伝播する。
- したがって光路は高P側へ曲がる。
